# spam_classifier
Naive Bayes spam classifier in python
----------------------------------------------------------------------------
Spam Classifier
----------------------------------------------------------------------------
1. pip install -r requirements.txt
2. To download NLTK stopwords 
>    	a. open python terminal
>	
>    	b. import nltk
>	
>   	c. nltk.downloader()
>	
>    	d. Enter 'stopwords' as identifier
>	
>    	e. quit and exit
	
3. Run Main.py

    python Main.py
